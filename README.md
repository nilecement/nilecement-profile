# موقع Nile Cement — نسخة ثابتة (Static) لاستضافة GitHub Pages

## المحتوى (محدّث)
**العربي هو اللغة الأساسية** على الرابط الرئيسي، **والإنجليزي تحت `/en/`**:
- **عربي (أساسي)**: `/`, `/about/`, `/services/`, `/products/`, `/quality/`, `/contact-us/`
- **English**: `/en/`, `/en/about/`, `/en/services/`, `/en/products/`, `/en/quality/`, `/en/contact-us/`

## آخر تحديثات
- **تبديل اللغة الأساسية**: العربي بقى في الجذر (nile-cement.com) بدل الإنجليزي، بناءً على طلبك.
- **الإيميل اتشال بالكامل** من الموقع (الهيدر، الفوتر، صفحة الاتصال، بيانات Schema) — التواصل بقى بالتليفون بس. أي حاجة تانية (واتساب، إلخ) تقوليلي أضيفها.
- **أيقونات السوشيال ميديا** رجّعتها بنفس ترتيب الموقع القديم بالظبط: TikTok، X، LinkedIn، Instagram، Facebook. لسه من غير روابط فعلية (زي ما كانت في الباكب الأصلي) — ابعتيلي اللينكات الحقيقية وأنا أضيفها.
- صورة المصنع الحقيقية في الـ Hero + صفحة "عن الشركة".
- صفحة منتجات شاملة بكل الجداول الفنية، وصفحة جديدة "الجودة والاعتمادات".
- **الأسعار مش منشورة على الموقع** — كل شيء عن طريق "اطلب عرض سعر".

## خطوات الرفع على GitHub Pages
1. اعملي Repository جديد على GitHub.
2. ارفعي كل المحتويات (`assets/`, `CNAME`, `sitemap.xml`, `robots.txt`, والمجلدات) لجذر الـ repository.
3. Settings → Pages → Branch: `main` والمجلد `/ (root)`.
4. اعملي DNS Records بتوجه لـ GitHub Pages:
   - 4 سجلات A على `@`: 185.199.108.153 / 185.199.109.153 / 185.199.110.153 / 185.199.111.153
   - سجل CNAME على `www` يشاور على `<username>.github.io`
5. فعّلي Custom domain = `nile-cement.com` و Enforce HTTPS من إعدادات GitHub Pages.

## ملحوظات لسه قايمة
- **فورم التواصل** شكلي بس (GitHub Pages مواقع ثابتة) — لو عايزاه يبعت فعليًا، اربطيه بـ Formspree أو مشابه.
- **لينكات السوشيال ميديا** لسه placeholder (#).
- **الخريطة** تقريبية بناءً على العنوان النصي بس.
- محتوى "نبذة عنا" و"خدماتنا" فيه صياغة عامة — راجعيها لو عندك تفاصيل إضافية.

